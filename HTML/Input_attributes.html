Input attributes

All these attribute will be used in "Input" tag.
1.value="Aditya" - It add value and display Aditya.
2.readonly- It will allow you to only read not write.
3.disabled- Same like readonly but light color.Use disabled instead readonly.
4.size="x"- Changes the size Increase or decrease.
5.rows="x" and cols="x"- It changes the rows and columns. It is only used in "textarea" tag.

HTML5 Form Input attributes

1.required- It makes a textfield as madatory.For ex "*" field.
2.novalidate- It removes a validation. Mostly added to "form" tag.
3.autofocus- Enable autofocus.
4.pattern- used from a websibe "HTML5 Pattern" type it on google.
5.placeholder- Use it you will understand.
