Table tags

Simple table

1."table" tag is used to add the table.
2."thead" tag is used to add headings of the table.it is optional.
3."tr" tag is used to add rows in table. It is used in "thead" or "tbody".
4."th" tag is used to enter data headings. it is written in "tr" tag.
5."tbody" tag is used to enter the data in table.
6."td" tag is used to enter the data od table.
7."col span" is an attribute used to combine two table column heading. It is used in "th" tag.
8."row span" is an attribute used to combine two rows in table body or table data.it is used in "td" tag.
