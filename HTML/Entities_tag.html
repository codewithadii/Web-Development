HTML Entities tags

If you want to display reserved charachers(<,>,!,#,etc) or if you want to display some special symbol you have to use html entities.
1.< - less than (<). 2.> - Greater than (>). 3.® - Register symbol. 4.© - Copy symbol. 5.₹ - Rupee symbol. Note - Use semicollen at the end of each entity(;).
