Color in CSS

In this tutorial of color in CSS we will learn in how many ways we can add color's in CSS (3 WAYS) : 1.Color with name.
2.rgb(x,x,x) - rgb means red,green,blue. We have pass parameter in palce of "x" and it will show a color by mixing these parameter.
3.#fff - by using # code.
