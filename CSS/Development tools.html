CSS Development tools tutorial

CSS coading is done in "style tag" which is also known as internal CSS.

In this tutorial we will learn about the following things :
1.color: -(text color) used to change the color of text.
2.Background-color: -As the name says used to change the background color.
3.font-size: - used to change the font size.
4.class="xyz" - changing color using class xyz, it is accessed by dot(.).
5.id="xyz" - changing color using ID.it is accessed by hash(#).
6.font-family: - used to change the writing of text.
